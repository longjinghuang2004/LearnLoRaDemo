### LoRaPlat 项目定位 (Project Positioning)

**一句话定义：**
LoRaPlat 是一个专为嵌入式系统设计的、**轻量级**、**高可靠**、**硬件无关** 的 **UART LoRa 链路层中间件 (Link Layer Middleware)**。

**核心价值 (Value Proposition)：**
它致力于解决廉价 UART LoRa 模组（如 ATK-LORA-01, Ebyte E32 等）在实际工程应用中的痛点，通过软件定义的方式，将一条“不可靠的物理串口线”升级为一条“可靠的逻辑数据链路”。

**关键特性 (Key Features)：**

1.  **可靠传输 (Reliability)**：
    *   内置 **Stop-and-Wait ARQ** (停等协议)，支持超时重传。
    *   实现 **ACK 确认机制**，确保数据必达。
    *   支持 **CRC16 校验**，杜绝空中噪声导致的数据篡改。

2.  **极致解耦 (Decoupling)**：
    *   **OSAL (操作系统抽象层)**：一套代码，无缝运行于 **裸机 (Bare-Metal)** 与 **RTOS (FreeRTOS/RT-Thread)** 环境。
    *   **Port (硬件接口层)**：仅依赖基础的 UART 收发与毫秒级时基，极易移植到 STM32, ESP32, MSP430, Linux 等任何平台。

3.  **异步并发 (Asynchronous & Concurrency)**：
    *   **非阻塞设计**：所有 API 均为非阻塞调用，绝不卡死主循环。
    *   **发送队列**：内置环形发送缓冲，自动处理半双工通信的时序冲突（如 ACK 避让），上层业务无需关心底层忙闲。

4.  **安全与管理 (Security & Management)**：
    *   **地址过滤**：支持逻辑 ID (NetID) 与 组 ID (GroupID) 过滤，实现点对点与组播通信。
    *   **指令鉴权**：内置平台级指令解析器 (CMD)，支持 Token 身份验证，可远程修改设备参数（如 ID、信道）。
    *   **加密钩子**：预留 Payload 加密接口，支持用户自定义算法。

**适用场景 (Use Cases)：**
*   **智能家居/楼宇**：节点众多，需要可靠控制（如开关灯必达）的场景。
*   **工业遥测**：环境恶劣，干扰大，需要数据完整性校验的场景。
*   **低成本组网**：使用廉价模组快速构建星型网络。

**非目标场景 (Non-Goals)：**
*   **LoRaWAN**：本项目不涉及 LoRaWAN 协议栈，不接入网关服务器。
*   **高速流媒体**：受限于 LoRa 物理特性，不适合传输音频/视频。
